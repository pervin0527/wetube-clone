## Global Router

/ -> Home  
/join -> Join  
/login -> Login  
/search -> Search

## User Router

/user/edit -> edit user  
/user/delete -> delete user

## Video Router

/videos/watch -> watch video  
/videos/edit -> edit video  
/videos/delete -> delete video  
/videos/comment -> comment on a video
/videos/comment/delete -> delete a comment
