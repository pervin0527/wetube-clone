## Global Router

/ -> Home  
/join -> Join  
/login -> Login  
/search -> Search

## User Router

/user/:id -> see user  
/user/logout -> log out  
/user/edit -> edit my profile  
/user/delete -> delete my profile

## Video Router

/videos/:id -> watch video  
/videos/:id/edit -> Edit video  
/videos/:id/delete -> delete video  
/videos/upload -> edit video
